# hexdump

## 一、hexdump 是什么

`hexdump` 是 Linux 下用于**查看二进制数据内容**的工具，可以把文件或标准输入的数据以十六进制、十进制或字符形式打印出来，常用于：

* 文件格式分析
* 协议调试
* 嵌入式存储、Flash、设备节点查看

---

## 二、最常用的用法

### 1. 以常用格式查看文件

```bash
hexdump -C 文件名
```

说明：

* 左侧是偏移地址（十六进制）
* 中间是十六进制字节
* 右侧是 ASCII 可打印字符

---

### 2. 只查看前 N 个字节

```bash
hexdump -C -n 64 文件名
```

适合查看文件头或 magic number。

---

### 3. 从指定偏移开始查看

```bash
hexdump -C -s 0x100 文件名
```

常用于跳过头部结构，查看指定区域。

---

## 三、按数据类型查看

### 按 16 位显示

```bash
hexdump -x 文件名
```

### 按十进制显示

```bash
hexdump -d 文件名
```

### 按字符显示

```bash
hexdump -c 文件名
```

---

## 四、查看标准输入

```bash
echo -n "ABC" | hexdump -C
```

可用于调试字符串、脚本输出或编码问题。

---

## 五、常见使用场景

* 查看二进制文件内容
* 分析 C 结构体内存布局
* 调试通信协议数据
* 查看 Flash / eMMC / 分区数据

---

## 六、小结

最推荐的组合：

```bash
hexdump -C -n 64 文件名
```

简单、直观、信息量足，是日常调试的首选。
